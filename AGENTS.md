# AGENTS.md

## プロジェクト概要

本プロジェクトはChrome拡張機能の開発プロジェクトである。

各種ドキュメントは `docs/` ディレクトリにあるので必ず確認すること。

## 実装に関する注意事項

- 日本語を第一言語としてコメントや回答などを行うこと
- 実装に当たってはDRY原則やSOLID原則といった原則を遵守すること
- わからない場合や推測を含む場合はその旨を必ず記載すること
- 機密情報の露出を始めとするセキュリティ面を必ず考慮すること
- 可能な限りテストを書き、コミット前に実行して成功を確認すること
- format と lint が導入されている場合は必ずコミット前に実行して成功を確認すること
- 不明点はまず最新の公式ドキュメントを確認すること
- インストールするライブラリやパッケージは現時点で最新のものを採用し、インストール方法や設定などは最新の公式ドキュメントを参照すること
